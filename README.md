# 단어 끝말잇기 (Word Replay)

## 🎮 게임 소개
한글 끝말잇기를 기반으로 한 교육용 게임으로, 다양한 테마의 스테이지와 수집 시스템을 갖춘 단어 학습 게임입니다.

### 주요 기능
- **다양한 테마 스테이지**: 10개의 특색 있는 테마별 스테이지
- **수집 시스템**: 게임 내 재화로 특별한 아이템 수집
- **자동 모드**: AI 도움말 기능으로 학습 보조
- **단어 추천**: 다음 단어 추천 시스템
- **점수 시스템**: 게임 성과 기록
- **사운드 시스템**: BGM과 효과음으로 게임 몰입도 향상

## 🛠 기술적 특징
- **데이터 관리**
  - CSV 파일 기반 단어 데이터베이스
  - PlayerPrefs를 활용한 사용자 데이터 저장
  - 효율적인 단어 저장 및 검색 시스템

- **구조적 특징**
  - 싱글톤 패턴의 매니저 시스템
  - 페이지와 팝업 관리 시스템
  - 모듈화된 UI 시스템

- **시각적 특징**
  - 동적 블록 시각화 시스템
  - 실시간 단어 연결 표현
  - 승리/패배 연출 효과

## 🎯 게임 진행
1. 스테이지 선택 메뉴에서 원하는 스테이지 선택
2. 주어진 시작 단어로 게임 시작
3. 이전 단어의 마지막 글자로 시작하는 단어 입력
4. 점수 획득 및 골드 수집
5. 모은 골드로 컬렉션 아이템 해금
6. 난이도가 올라가는 스테이지 진행
